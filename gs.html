UI/UX Design

Master the art of creating intuitive user interfaces (UI)...

6 Weeks Intermediate

By Emily Johnson

Description

Description This course introduces the fundamentals of User Interface (UI) and User Experience (UX) design, guiding learners through the complete design process — from user research to wireframing and prototyping. You'll explore design principles, tools like Figma or Adobe XD, and learn to create responsive, user-friendly interfaces for web and mobile applications. Whether you're a beginner or aspiring designer, this course helps you build a strong foundation and a professional design portfolio.

Course Outcomes

  • 🎯 Understand UI and UX design principles
  • ✏️ Design user-centric wireframes, mockups, and prototypes
  • 🖥️ Create responsive designs for web and mobile
  • 🛠️ Use Figma, Adobe XD, Sketch
  • 🔍 Apply usability testing techniques
  • 💡 Solve real-world problems with design thinking
  • 🚀 Showcase a complete UI/UX project in a portfolio
